# Session Notes

Prep notes for each played session. These are the **source documents** for building interactive pages in `gm_sessions_site/`.

## Workflow

1. **Before session** — Draft or update `Session_N_<Title>.md` using the session-notes skill.
2. **During/after session** — Add player decisions, new NPCs, world changes, and thread updates.
3. **Build session site** — Convert finalized notes to `gm_sessions_site/sessionN.html` (+ location/fear pages as needed).
4. **Sync canon** — Push new facts back into `02_Regions/`, `09_NPCs/`, clocks, and faction trackers.

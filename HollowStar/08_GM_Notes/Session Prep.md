# Session Prep Quick Reference

## Before Session

### Check These
- [ ] Which factions are players allied with?
- [ ] Where is the approaching star countdown?
- [ ] What's the state of Lumencrest/Endless Eye?
- [ ] Which mysteries are players investigating?
- [ ] What consequences from last session?

### Prepare
- 3-5 NPCs (names, goals, secrets)
- 1-2 combat encounters (if likely)
- 2-3 social/political scenes
- Environmental descriptions
- Faction reactions to player actions

## During Session

### Opening
- Recap with player input
- Establish scene/location
- Present immediate situation

### Core Principles
- **Say yes, or yes but...**
- **Spotlight each player**
- **Make failures interesting**
- **Telegraph danger**
- **Honor player agency**

### Pacing
- Mix action, exploration, social
- Build to climax
- End on choice or cliffhanger

## After Session

### Note
- Player decisions and consequences
- NPC relationships changed
- World state changes (power levels, water, etc.)
- Mysteries discovered/deepened
- Faction reputation shifts
- Countdown clock advances

### Prep for Next
- What will factions do in response?
- Which threads to pull?
- What consequences manifest?

## Improv Tools

### Need NPC Name?
- Lumencrest: Latin-ish (Cassius, Valeria)
- Endless Eye: Nature-themed (Dewleaf, Stoneskipper)
- Northern: Norse-ish (Bjorn, Freya)
- Verdant Crescent: Elvish (Aelindra, Theron)

### Need Complication?
- Power flickers at crucial moment
- Rival faction arrives
- Transformed creature appears
- NPC has hidden agenda
- Environmental hazard
- Timer starts ticking

### Need Clue?
- Fragmentary ancient text
- Contradictory witness accounts
- Physical evidence (strange material)
- NPC knows something but wants payment
- Dreams/visions
- Stars behaving strangely
